A.  In AndroidManifest.xml check for 'uses-permission AD_ID' and if found then
    <uses-permission android:name="com.google.android.gms.permission.AD_ID" tools:node="remove"/>
    <uses-permission android:name="android.permission.ACCESS_ADSERVICES_AD_ID" tools:node="remove"/>

B.  Add 'signingConfigs' above to the 'buildTypes' in 'build.gradle.kts'(app) 
    and also use 'isMinifyEnabled = true' and 'debugSymbolLevel' if possible , given as below
	signingConfigs{
        	create("release"){
            	keyAlias = "digivahan"
            	keyPassword = "digivahan@android"
            	storeFile = File("C:\\Users\\aadit\\StudioProjects\\digivahan-android\\app\\src\\main\\digivahan.txt")
            	storePassword ="digivahan@android"
        	}
    	}

    	buildTypes {
        	release {
            	isMinifyEnabled = true
            	proguardFiles(
                	getDefaultProguardFile("proguard-android-optimize.txt"),
                	"proguard-rules.pro"
            	)
            	ndk {
                     debugSymbolLevel = "SYMBOL_TABLE"
            	}
            }
        }
